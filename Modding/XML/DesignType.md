A DesignType is an XML structure that defines game data for in-game Types.

- AdventureDesc
- DockScreen
- EconomyType
- EffectType
- Image
- ImageComposite
- ItemTable
- ItemType
- MissionType
- NameGenerator
- OverlayType
- Power
- ShipClass
- ShipClassOverride
- ShipTable
- Sound
- Sovereign
- SpaceEnvironmentType
- StationType
- SystemMap
- SystemTable
- SystemType
- TemplateType
- Type
